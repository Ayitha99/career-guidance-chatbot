# 🎓 Career Guidance Chatbot

An AI-powered Career Guidance Chatbot that helps students and professionals explore career paths, identify required skills, and follow structured learning roadmaps.

Built using **Python, Streamlit, and Groq API**, this chatbot provides real-time, personalized career advice through an interactive chat interface.

---


---

## 📌 Features

* 💬 Interactive AI Chat Interface
* 🎯 Personalized Career Suggestions
* 🧠 Skill Recommendations by Domain
* 🗺️ Step-by-Step Learning Roadmaps
* ⚡ Quick Career Option Buttons (Software, AI, Govt Jobs)
* 🔁 Chat Memory using Session State

## 🧠 How It Works

1. User enters a career-related query
2. Chat history is maintained using Streamlit session state
3. The query is sent to the Groq LLM API
4. The AI generates a response with career advice
5. Response is displayed in a user-friendly format
